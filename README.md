TextUtils is a simple yet powerful text analysis tool built with React. It provides an efficient way to analyze and manipulate text, offering a variety of features to streamline text editing tasks.

Features:
Copy Text: Quickly copy the processed text to your clipboard with a single click.
Remove Extra Spaces: Eliminate unnecessary spaces in your text for cleaner formatting.
Text Capitalization: Convert text to uppercase or lowercase easily.
Dark Mode & Light Mode: Toggle between dark and light themes for a more comfortable user experience.
Real-Time Updates: See changes to your text instantly as you use the app.

Usage:
Input your text in the editor.
Choose the desired function (e.g., remove spaces, capitalize text).
View the processed result immediately.

Technologies Used:
React: Frontend framework for building the user interface.
JavaScript: Core scripting language.
CSS: Styling for themes and layouts.
